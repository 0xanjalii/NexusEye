# 🌌 NexusEye: Advanced Solana Analytics Dashboard

![NexusEye Hero](./public/readme-hero.png)

NexusEye is a high-performance, sci-fi themed analytics dashboard for the Solana blockchain. Powered by the **Birdeye API**, it provides real-time insights into token price action, whale movements, smart money tracking, and detailed market metrics.

## ✨ Key Features

- **🚀 Real-time Price Action**: Interactive time-series charts for any Solana token using Recharts.
- **🐋 Whale Tracker**: Live monitoring of large-scale transactions to spot market movers.
- **🧠 Smart Wallet Insights**: Track the most profitable "Smart Money" wallets in the ecosystem.
- **📊 Advanced Token Metrics**: Deep dive into liquidity, volume, and holder statistics.
- **⚡ Live Trade Feed**: Real-time visualization of swap activity across the network.
- **🖥️ Sci-Fi UI/UX**: Premium, high-contrast interface designed for power users and traders.

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript
- **Bundler**: Vite
- **Styling**: Vanilla CSS (Custom Sci-Fi Design System)
- **Animations**: Framer Motion
- **Data Visualization**: Recharts
- **Icons**: Lucide React
- **API**: Birdeye Data Service

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Birdeye API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/nexuseye-dashboard.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the root directory and add your Birdeye API key:
   ```env
   VITE_BIRDEYE_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
